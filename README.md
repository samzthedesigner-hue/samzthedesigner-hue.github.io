# Torch // Local Core

**100% Offline AI Code Generator PWA**

Torch runs a 1.5B coding model directly in your browser using WebGPU. No API keys. No internet required after first load. Build full-stack apps and download them as a .zip

Built by: [@samzthedesigner.hue](https://github.com/samzthedesigner.hue)

## 🚀 Live Demo
**[https://samzthedesigner.hue.github.io/torch/](https://samzthedesigner.hue.github.io/torch/)**

Tap "Install to Android" on Chrome to install as a PWA

## ✨ Features
- **Fully Offline**: Downloads Qwen2.5-Coder-1.5B and runs on-device
- **Multi-File Generation**: Outputs React, Node, Python, Flutter, Java projects
- **PWA + APK Ready**: Installable on Android. Works offline
- **Project Exporter**: Compiles all files into a downloadable .zip
- **Dark Terminal UI**: Built with React + Tailwind + Vite

## 🛠️ Tech Stack
`React` `Vite` `TailwindCSS` `WebLLM` `@mlc-ai/web-llm` `JSZip` `PWA` `Capacitor`

## 📦 How to Use
1.  Open the Live Demo link above
2.  Wait for "AI Intelligence Core" to load to 100%
3.  Describe your app in the prompt box. Be detailed
4.  Select your tech stack
5.  Click "Compile & Package Distribution Archive"
6.  Download your project .zip

## 📱 Install on Android
1.  Open the link in Chrome
2.  Tap the 3 dots > `Install app`
3.  Or tap the "Install to Android" button inside Torch
